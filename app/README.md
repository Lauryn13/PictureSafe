# PictureSafe - Foto Tresor

[Link zum Github-Projekt](https://github.com/Lauryn13/PictureSafe)

Diese App wurde für das Modul Mobile Systeme entwickelt.
Es handelt sich dabei um einen Steganographie Tresor, bei dem man mit hilfe von Bildern Dateien versteckt in einem oder mehreren Bilder speichern kann. Zusätzlich abgesichert durch Signaturen, Kompression und AES-Verschlüsselung ist es sicher vor jedem der ungewollt diese Datein sucht.<br>

Eine APK-Datei wurde im Ordner APK-Builds/app-debug.apk hinterlegt. Andernfalls kann die App auch über den Import des Projektes in Android Studio installiert werden.<br>

Viel Spaß beim verstecken wichtiger Datein in den süßen Katzenfotos :)<br><br>
